주니어 코스
영어 학원 안가도 되는! 프로그램

우리 아이가 부담없이 영어에 흥미를 느끼고, 스스로 말하는 힘을 키울 수 있도록!
영어의 4 parts (듣기, 읽기, 말하기, 쓰기) 완성 프로그램!

이런 분들에게 추천해요

언어 학습은 어릴수록 좋습니다!
01. 학원에 가지 않고도
체계적으로 영어를 배우고 싶은 어린이

"학원을 꼭 다녀야 하나?"
고민하는 부모님과 아이들을 위한 선택!

- 등하원 걱정 없이 집에서 편하게 공부하고 싶은 아이
- 대상 : 초등학생
- 학원에 가기 싫어하지만 영어는 배우고 싶은 아이

"엄마, 오늘 학원 안 가면 안 돼?"
하지만 영어는 잘하고 싶어 해요!
02. 회화 실력을 키우고 싶은데
연습할 기회가 부족한 학생

"배운 영어,
직접 말해볼 기회가 부족해요!"

- 교과서 문법 공부는 하지만 실제로 말해볼 기회가 없는 아이
- 머릿속으로는 문장이 떠오르지만 입 밖으로 나오지 않는 아이
- 원어민처럼 말하고 싶은데 연습할 상대가 없어 아쉬운 아이

"배운 거 기억은 나는데,
어떻게 말해야 할지 모르겠어!"
03. 영어를 재미있게
익히고 싶은 주니어 학습자

"지루한 공부 NO!
재미있게 배우고 싶어요!"

- 단순한 문법 공부보다 이야기와 캐릭터로 배우고 싶은 아이
- 애니메이션, 게임, 롤플레이 등으로 신나게 공부하고 싶은 아이
- "재미있어야 오래 기억한다!" 즐겁게 배워야 집중하는 아이

"공부 같지 않은데,
영어가 점점 쉬워지는 기분이야!"
목표

최대한 한국에서 영어 노출이 많이 될 수 있도록 프로그램 구성
1. 재미있는 "동화" 수업 + 온라인 사이트 활용하여 Input 극대화
2.재미있는 수업 지향
3. 학습 시간 효율적 관리 - 학원 가고 오는 시간 활용
4. 탄탄한 학습 관리 시스템
5. 이벤트 : 학습 동기 부여

개요

대상 : 초등학교 저학년 ~ 고학년 (AR 0.5~ AR 5.0)
구성 : 회화 패턴 학습 + 스토리 기반 Input & Output
1. 리딩 2. 단어(녹음) 3. Listen & Repeat 4. Storytelling

Let's Go 시리 즈 + 엄마표 영어 ( 온라인 영어 사이트 ) 활용

교재

Let's go 시리즈 + 라즈키즈 온라인 영어 도서관

  • 영어 문장 패턴 학습
    기본적인 문법 학습
  • 라즈키즈란?
    미국, 캐나다 등 영어권 국공립학교에서 정규 수업시간에
    활용하는 온라인 영어 독서 프로그램입니다.

라즈키즈

  • 영어 리딩 유창성 및 정확성 동시 향상 프로그램
  • 총 29개 레벨, 3,000여 권 이상의 다양한 스토리북 탑재
  • 픽션, 논픽션, 과학, 역사, 스포츠 등 광범위한 주제로 배경지식 확장
  • 듣기, 읽기, 녹음하기, 퀴즈풀기 구성의 읽기 학습 프로세스
  • PC, 스마트폰, 태블릿 기기 상관없이 24시간 자유롭게 자기주도 학습 가능
  • 1. 북미 교사 평가 1위 프로그램으로 국내에서 미국, 캐나다 국공립학교와 동일 수준 원서 리딩 가능
  • 2. 미국 전,현직 교사 및 전문가가 직접 집필한 원서
  • 3. 약 3천여 권의 픽션, 논픽션 원서들을 레벨별, 주제별, 시리즈별로 학습 가능
  • 4. 관리자의 객관적인 리포트 확인 시스템 구축으로 독서 기록 분석 가능
  • 5. 인터넷이 가능한 환경이라면, 언제 어디서든 PC, 모바일 기기로 온라인 학습 가능
  • 6. 국내 교사들이 꼽는 최고의 가성비 온라인 영어 독서 프로그램
